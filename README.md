# 🤖 Multi-AI Account Manager

A free, browser-based dashboard to manage multiple AI accounts (Claude, Codex, Grok, etc.), track usage timers, switch between accounts, and organize client projects.

![Dashboard Preview](preview.png)

## ✨ Features

- **Multiple AI Accounts**: Add unlimited accounts (Claude, Codex, Grok, ChatGPT, Gemini, etc.)
- **Usage Timer Tracking**: Real-time countdown per account (in IST)
- **Project Management**: Create projects (Ongoing/Completed/Future) and link accounts
- **Auto-Switch Suggestions**: Know when to switch accounts based on timer limits
- **Password Protection**: Simple security for your data
- **Data Export/Import**: Backup and restore data as JSON
- **Responsive Design**: Works on desktop and mobile

## 🚀 Quick Start

1. Download all 3 files: `index.html`, `style.css`, `app.js`
2. Open `index.html` in your browser
3. Set a password on first use
4. Start adding accounts and projects!

## 🌐 Deploy to GitHub Pages

1. Create a GitHub repository (e.g., `ai-account-manager`)
2. Upload the 3 files
3. Go to Settings → Pages → Enable GitHub Pages
4. Access via: `https://yourusername.github.io/ai-account-manager/`

## ⚠️ Note

This is a client-side only app. Data is stored in your browser's localStorage. Use Export to备份 your data regularly.
